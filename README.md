# SARTHI – Campus Transit System (CTS)

SARTHI is a web-based Campus Transit System designed to digitize and simplify campus bus information management.  
The project focuses on improving clarity, accessibility, and communication around daily bus routes, schedules, and announcements within an educational campus.

---

## Project Overview

In many campuses, bus schedules and route updates are still managed manually using physical notice boards. This often leads to confusion, missed information, and delays, especially when updates change frequently.

SARTHI addresses this problem by providing a centralized, easy-to-use web interface where students and staff can access transport-related information anytime, without depending on physical locations.

---

## Key Objectives

- Centralize campus bus route and schedule information  
- Reduce dependency on manual notice boards  
- Improve visibility of announcements and daily updates  
- Provide a clean and intuitive user experience for both users and administrators  

---

## User Roles

### Admin
- Manage bus routes and assignments  
- Update daily bus status (on-time, delayed, cancelled)  
- Post transport-related announcements  
- Access a protected admin panel for operations  

### User (Student / Staff)
- View available bus routes and schedules  
- Check bus stop timings  
- Read important transport announcements  
- Access information from mobile or desktop devices  

---

## Core Features

- Route-based bus listing with search and filters  
- Detailed stop timings for selected buses  
- Announcement section for urgent and general updates  
- Separate admin and user views  
- Responsive layout for mobile and desktop screens  

---

## Design Approach

- Simple and clean UI focused on readability  
- Sidebar-based navigation for clarity  
- Modular sections for routes and stop tracking  
- Smooth transitions between views for better UX  

---

## Current Status

- Requirement analysis completed  
- User roles and core features defined  
- Initial screens and component structure implemented  
- UI and interaction flow under active development  

---

## Scope

SARTHI is designed as a scalable foundation that can be extended with additional features such as analytics, historical tracking, or backend integration in future iterations.
